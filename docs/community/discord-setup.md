## 👥 Roles & Permissions

### Role Hierarchy (Top to Bottom)

#### 🏆 Server Owner
- Full administrative control
- Can manage all aspects of the server

#### 🔧 Admin
- Manage channels and roles
- Moderate content
- Manage server settings
- Access to all channels

#### 🛡️ Moderator
- Moderate messages and users
- Manage specific channels
- Handle disputes and issues
- Access to moderation tools

#### 🎯 Core Contributor
- Access to development channels
- Can create threads
- Priority support
- Recognition for contributions

#### 🚀 Active Contributor
- Access to most channels
- Can participate in discussions
- Recognition for regular contributions

#### 👨‍💻 Developer
- Access to technical channels
- Can ask questions and get help
- Participate in development discussions

#### 🎓 Community Member
- Basic member permissions
- Access to general channels
- Can participate in discussions

#### 🆕 New Member
- Limited access to channels
- Read-only in most channels
- Can introduce themselves

## 📋 Server Rules

### General Rules
1. **Be Respectful** - Treat everyone with respect and kindness
2. **No Harassment** - Harassment of any kind is not tolerated
3. **Stay On Topic** - Keep discussions relevant to the channel
4. **No Spam** - Avoid excessive posting or self-promotion
5. **Follow Discord ToS** - Abide by Discord's Terms of Service

### Technical Rules
1. **Use Appropriate Channels** - Post in the correct channels
2. **Search Before Asking** - Check existing discussions first
3. **Provide Context** - Include relevant details when asking for help
4. **Be Patient** - Allow time for responses
5. **Share Knowledge** - Help others when you can

### Content Guidelines
1. **No NSFW Content** - Keep content appropriate
2. **Respect Copyright** - Don't share copyrighted material
3. **No Personal Information** - Don't share personal details
4. **Constructive Criticism** - Provide helpful feedback
5. **No Political Discussions** - Keep politics out of technical discussions
